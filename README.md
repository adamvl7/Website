# Adam Le Portfolio

Interactive personal portfolio built with **Next.js + TypeScript**, featuring a custom space-themed UI, animated project solar system, resume viewer, and contact experience.

## Live Sections

- **Home**: Intro, animated multilingual greeting, about section, skills, social links
- **Experience**: Work/experience timeline
- **Projects**: Interactive orbiting planets for each project with modal details
- **Contact**: Interactive galaxy canvas with click-to-explode/reform effect
- **Resume**: In-site PDF resume viewer + download

## Tech Stack

- **Framework**: Next.js 14
- **Language**: TypeScript
- **UI**: React, custom CSS, Tailwind (available in project config)
- **Animation**: requestAnimationFrame + CSS animations
- **Deployment**: Vercel
